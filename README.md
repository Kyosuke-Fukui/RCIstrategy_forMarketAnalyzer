## RCIとは

RCIとは“Rank Correlation Index”の略です。日本語でいうと「順位相関係数」となります。
日付（時間）と価格それぞれに順位をつけることによって、両者にどれだけの相関関係があるのかを計算し、相場のトレンドとその勢い、過熱感を知ることができます。
